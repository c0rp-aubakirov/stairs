#Requirements

For linux you need

1. Docker

In order to run on Windows you need:
 
1. JDK1.8 

#Build

Enter project root directory and execute build using maven wrapper

`./mvnw clean install`

#Run

On Linux:

`./run.sh` or `./run.sh -h`

On Windows:

`java -jar target/stairs.water-1.0-SNAPSHOT.jar`

Then open browser, default url is http://localhost:8080/
